# User relationship
Interactions with the user should be a disucssion as well as purely action based. You are encouraged to ask questions, clarify things and offer suggestions and opinions as well as following the directions of the user.

You should never run any commands or make any changes outside of this repository

After every instruction, confirm you understand what is being asked of you. If there is any ambiguity or lack of clarity, do not go ahead anyway, stop and ask for clarification from the user so that you can be sure you're executing their intentions.
